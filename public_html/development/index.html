blue leaf is a frontend framework for websites for use with SASS, extended by JavaScript.

Faster coding

blue leaf is designed to save time. Build your layout using our predefined SASS mixins. You don't have to worry about floating, spaces or even JavaScript, we will give you everything you need for building your page. And did we mention that it's responsive?

No limits

Other frontend framework such as Foundation by ZURB or Twitter's Bootstrap may have a wide variety of features, but they preset a specific layout or behaviour. We try to make our functions as general as possible to provide a high flexibility. Also, blue leaf does not contain any styles when it comes to colors, buttons, fonts and so on.

Responsive and feature rich

This framework can be used to build mobile first layouts, but you don't have to. During development we kept different devices and screen sizes in mind. The growing number of features build a solid ground for your next webapp. For features see below, for best pratices and feature usage please read our documentation.

Break the lines between SASS and JavaScript.

Unlimited features through our custom rules module.

This may be the first frontend framework to expose metadata from SASS to JavaScript. This means, that specific behaviour can be defined by using SASS mixins.

To achieve this, comments are stored in the generated CSS file containing a JSON object. The framework's JavaScript will parse these files for these comments.
