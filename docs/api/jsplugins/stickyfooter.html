Sticky Footer¶

JavaScript¶

Plugins.fn.stickyfooter(scrollarea)¶

Sticks the footer to the bottom of the scroll area if the content of the scroll area is too small to fill the scroll area. The current element is the footer.

Use “html” as scrollarea to use the main scroll area.

Arguments:
  • scrollarea (string) – jQuery selector.
Returns:

An object providing enable and disable functions.
