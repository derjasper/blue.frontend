Sticky¶

This plugin enables sticky elements. They can stick to a scroll area’s edges and can be hold inside a parent element.

JavaScript¶

Plugins.fn.sticky(directions, scrollarea_sel, container_sel, sticky_class)¶

Makes the current element sticky.

Add margin and z-index to the element’s sticky_class to define offsets and stacking behaviour.

Arguments:
  • directions (object) – A map containing boolean values for top and bottom keys. Indecates at which edges of the scroll area the element will stick.
  • scrollarea_sel (string) – jQuery selector for the scroll area (it looks for the first parent matching the selector). Use "viewport" to use the window.
  • container_sel (string) – jQuery selector for the parent element to stay in (it looks for the first parent matching the selector). The element will stop sticking if it would leave its container.
  • sticky_class (string) – A class to be applied when sticked.
Returns:

An object providing enable and disable functions.
