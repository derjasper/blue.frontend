blue.frontend is a frontend framework for websites for use with SASS, extended by JavaScript.

Faster coding

blue.frontend is designed to save time. Build your layout using our predefined SASS mixins. You don't have to worry about floating, spaces or even JavaScript, we will give you everything you need for building your page. And did we mention that it's responsive?

No limits

Other frontend framework such as Foundation by ZURB or Twitter's Bootstrap may have a wide variety of features, but they preset a specific layout or behaviour. We try to make our functions as general as possible to provide a high flexibility. Also, blue.frontend does not contain any styles when it comes to colors, buttons, fonts and so on.

Responsive and feature rich

This framework can be used to build mobile first layouts, but you don't have to. During development we kept different devices and screen sizes in mind. The growing number of features build a solid ground for your next webapp. For features see below, for best pratices and feature usage please read our documentation.

Break the lines between SASS and JavaScript.

Unlimited features through blue.frontend's JavaScript plugin API.

This may be the first frontend framework to expose metadata from SASS to JavaScript. This means, that specific behaviour can be defined by using SASS mixins.

To achieve this, comments are stored in the generated CSS file containing a JSON object. The framework's JavaScript will parse CSS files for these comments.

Those JavaScript plugins are fully responsive and are nearly transparent to the developer.

Basic Layout

  • Container
    advanced sizing and positioning of elements
  • Fluid grid
    multi-column layouts
  • Gutter
    consistent spacings
  • Alignment
    aligning content just works
  • Lists and Menus
    no worries about accurate layouts

JavaScript plugins

  • Sticky Elements
    elements that stick at the top of the viewport
  • Offcanvas
    show less important elements on demand
  • Trigger
    define click and mouseover handlings from SASS
  • Scroll position
    trigger an event when an element is scrolled to
  • Smooth scrolling
    smooth scrolling on jumplinks

Framework features

  • Breakpoints
    responsive design
  • Custom rules
    the missing bridge between SASS and JS: extend CSS rules using blue.frontend's JavaScript plugin API
  • Shorthand CSS rules
    don't have to define often-used css classes

Please refer to our documentation for more details and a full list.
